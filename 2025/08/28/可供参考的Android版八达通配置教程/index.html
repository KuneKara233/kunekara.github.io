可供参考的Android手机八达通通用配置教程

发表于 2025-08-28 更新于 2025-08-29 分类于 教程
我真的不想再丢八达通了！

缘起

由于本人寒暑假有空经常入港，八达通成为必要的随身物品。虽说香港现在移动支付已经相当普及，甚至现在MTR大部分闸机都已经接入交通联合，直接拿内地的交通卡也能刷，有些小型巴士以及地方相对偏僻的一些苍蝇馆子依然是只能使用八达通的。此外由于香港糟糕的通信基础设施建设，移动支付在很多时候会掉链子，尤其是在使用香港SIM卡/人流量非常大的时候（猜你想看：港迪闭园时的迪士尼站），运气差一点的话还会出现在闸机口想打开二维码，结果卡了半天还没出现，最后还得挨叼的情况（对吧，香港很多老妈子素质确实低得不行）。
港铁站申领的八达通，都是租赁的不挂名 八达通，押金50HKD（是用来避免你出不了站的，如果余额不足，就会再从里面扣钱），所以丢了就是50HKD连着卡里的余额一块全部没掉，也无法挂失。迄今我已经在香港丢失了三张八达通了，深感心痛。为了克服容易丢失的弊端，我开始思考如何将八达通转移到手机上，于是就有了这一份教程。

教程

注意：本教程面向Android手机用户。iOS用户可以直接在Apple Pay中开八达通卡，操作流程比Android简单，具体过程请另行搜索。本文默认您具备自行搜寻/访问相关必要软件的能力。

八达通公司官方网站目前显示能有效支持Android版八达通的手机品牌/型号有Google Pixel，华为（华为钱包）1,，三星（Samsung Pay）以及在这里所列出的手机型号。原理上来说，只要是个Android并且带有NFC芯片的手机都可以根据这份教程进行操作。

演示机器

realme GT（国行，realme UI 5.0，Android 14，安全补丁Apr 5, 2024）

这也是我（截止到写这篇文章时）的主力机，如果以后换到iOS，我还会再在这里更新从Android迁移八达通到iOS的教程。

准备工作

需要以下这些内容：

一台支持NFC的Android手机

Google账号

一张不记名的八达通2

能访问香港互联网的手段3

一张香港手机卡（没有也行，有了注册更简单）

一张银行卡（内地/香港都可以）

操作步骤

如果有可能，以下操作请尽量在香港网络环境下进行，以提高操作的稳定性和成功率。

注册八达通

  1. 安装Octopus App，Google Wallet和银联云闪付4。这三样东西都可以在Google Play找到。
  2. 登录Octopus App并注册账号，如果存在报错（不支持），则再安装[Play Integrity API Checker](play integrity api checker - Android Apps on Google Play)，这个同样也能在Play商店找到。安装后运行检查，设备至少需要达到第二个指标（MEETS_DEVICE_INTEGRITY）才可以继续进行。如果你的设备没有达到第二个指标，那么可以通过进行系统更新，或者直接到APKPure之类的网站找到Play商店的最新包体并直接安装。同时，顺带把你的Google Wallet更新一下。
  3. Octopus App的账号，一般来说+86和+852都是可以注册的，如果你实在没有访问香港网络的手段，那么可以尝试使用移动数据进行访问，实在不行的话就建议肉身访港进行。购买+852的手机卡也很简单，现在香港卡的不是很死，线下随便找一个就能买，保证能收取短信就可以。将你的银行卡绑定到银联云闪付上，并在发出这张银行卡的银行App启用境外支付。
  4. 进入八达通主页之后选择“Android版八达通”字样旁边的+号，按照图片提示把八达通贴在手机的NFC感应区上。然后会要你同意一大堆协议什么巴拉巴拉的，直接过就行。复制八达通卡之前最好确保卡内有一定的余额，这样的话复制完如果急用还能有一些余裕。当然，如果没有八达通也没关系，Octopus App是支持虚拟开卡的，但八达通卡虚拟开卡需要先充值，具体步骤请参考“给八达通充值”5。
  5. 加入完成之后会显示“新增至Google钱包”，直接顺着一直点确认就可以。然后将手机的NFC默认支付方式修改为Google Pay，这样就配置完成了。需要过闸/乘车/结账的时候，直接将NFC感应器碰到读卡器上面就可以了。如果还是担心刷卡失败，变成支付宝碰一碰/手机钱包里面的交通联合，可以先把Google Wallet打开，点开八达通到有“靠近读取设备”或者类似含义的字样的时候，就可以直接刷了。

给八达通充值

主页下方中间有一个八达通标志的按钮“增值八达通”，点进去，选择你想要增值的卡，在“增值方式”里面选择银联云闪付。如果是开卡时充值，那么也是基本一样的操作步骤。需要注意的是，使用银联云闪付时，用来支付的银行卡必须是可以进行境外交易的，而境外交易功能需要另行开启（一般来说，直接在相应的银行App里面搜素“境外交易”就可以找到），否则会导致支付失败。除了给虚拟八达通充值之外，Octopus App也可以像羊城通小程序一样通过NFC给实体卡充值，具体操作步骤大致相同，只是需要将卡贴在NFC感应区，此处不再赘述。

自定义八达通

Octopus App提供卡面商店，其中提供许多卡面。新开卡的用户拥有优惠，可以在里面免费获得一部分卡面（当然，优惠只能享受一次），其他卡面则是需要另行购买，一般都是在50HKD以下。具体操作很简单，此处只是介绍有这么一个功能。 其实里面可以买到的卡面很少

配置扣款提示

使用虚拟八达通付费的时候，除非你打开Google Wallet再刷卡，否则扣款是不会有任何提示的。为了避免麻烦。可以转到设置将锁屏显示等权限授权给Octopus App，以此来减少不必要的麻烦。一般是长按Octopus App→应用信息→权限管理→将锁屏和桌面相关的权限打开。此步操作因系统而异。

[1]华为钱包只测试过EMUI/HarmonyOS 4.0及以前的版本，纯血鸿蒙尚未经过测试。不过香港应该也没多少人用纯血鸿蒙吧

[2]八达通除了不记名的租赁版以外，还有记名/个人八达通、小童八达通、学生八达通、乐悠咭等版本。后面这些基本都是需要提供香港户口/身份证明文件的，既然都能来看这个教程了，那说明大概率您也没有香港户口。不记名八达通操作起来最为简单便捷，比上面这些需要验证身份信息什么的能少不少麻烦。关于内地发售的交通联合版八达通，由于该种八达通采用双芯片方式（一个是人民币钱包，一个是港币钱包），而且Google Pay不支持内地交通联合，所以有可能会出现无法复制/复制后无法正常工作的情况。注意：不要想着利用上面那些特殊的八达通来获取搭车优惠，违法使用八达通是会受到刑事检控的。

[3]可以翻墙，也可以肉身到香港访问，随便找个免费公共WiFi就行。如果是用流量，那么记得看清要购买可以使用Google服务的流量包。

[4]如果你有香港搞到的银行卡也可以，这样操作更简单。内地银行卡只要不是虚拟卡都可以（不会有人用社保卡来绑定云闪付吧？），甚至是校园银行卡也可以。银联云闪付推荐直接利用手机自带的应用商店/前往银联官网下载，如果使用的是GP包体以后更新可能会有点麻烦。

[5]虚拟开卡一般需要先充值至少100HKD，这一点和线下直接搞实体的不记名八达通是一样的。
